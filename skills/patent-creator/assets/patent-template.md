## 4. Detailed Description of the Invention (Required - CRITICAL SECTION)

Describe your invention and its impact in detail, including:

### 4.1 Core Ideas
- Concepts
- Methods
- Structure
- Process steps to achieve the goals

### 4.2 Supporting Materials
- Drawings
- Flow charts
- Simulation results
- Experimental data
- Relevant parameters (shapes, dimensions, functions, etc.)

### Guidelines for This Section:

1. **Use diagrams**: Include flowcharts, system architecture diagrams, product structure diagrams to detail the implementation (methods, steps, systems, devices, modules, components, structures, control processes, etc.)

2. **Detail level**: Should be sufficient for a person skilled in the art to implement the invention

3. **Focus on novelty**: Emphasize technical features that differ from prior art, explain how they are implemented, and describe corresponding technical effects

4. **For algorithm patents**: Provide at minimum:
   - One **method embodiment** (e.g., "A method comprising steps S1, S2, and S3")
   - One **system embodiment** (e.g., "A system comprising modules A, B, and C")

5. **Multiple embodiments**: Consider variations:
   - If invention includes steps A, B, and C, can it work with only A and B?
   - Can it also work with steps A, B, C, and D?
   - If yes, document each as a separate embodiment

6. **Terminology**:
   - Provide full names for English technical terms on first use
   - For terms without accurate translations, provide reasonable Chinese translations based on experience
